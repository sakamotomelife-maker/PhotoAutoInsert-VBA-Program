# エクセル内に写真を一括で挿入するツール
![画面例](docs/screenshot1.png)

## 概要
- エクセル内の商品リストを写真が入るよう整形し、写真を順番に挿入するツールです。
- 100商品の写真を撮り、圧縮してエクセルに挿入、エクセル内で写真を移動する工程を大幅に削減します。

## 主な機能
- 指定したエクセル（商品リスト）の高さ・幅を調整
- スマホで撮った写真を撮った順で商品リストの上からいれていく

## 使用方法
- Sampleファルダ内の各データをダウンロード（公開用に作り直しています。）
- 選択ボタンから対象エクセル、対象画像フォルダを選択
- 整形ボタンから対象エクセルの列幅等を変更<br>
  ![画面例](docs/screenshot3.png)
- 圧縮ボタンから写真を一括で圧縮<br>
  ![画面例](docs/screenshot4.png)
- 挿入ボタンから画像をエクセルファイルに挿入<br>
  ![画面例](docs/screenshot2.png)

## 技術ポイント
- 写真ファイルをPNGからJPGに統一
- 横長の写真は縦長に回転（逆に回転する可能性もあるがご愛敬)
- 写真の一括圧縮
- ファイル名順に写真をエクセルへ挿入
